# Webkins
----------------------------------------------------------------------------------------------------------
# Overview
Web application targeted towards animal lovers that offers several features to learn. The goal is to use
different animal live cams to implement a geoguessr game.
An interactive zoo web application using python.
# Motivation
A mutual love of animals and interest in spotlighting different zoos.
# Features
Geoguessr-type game
● Displays picture of an animal and the user needs to place a pin on a map closest to where they
think it originates
● Calculates a score based on distance from correct location
● Provide a small educational blurb/fun facts about each animal
● Several game modes (timed, daily challenge, live multiplayer versus)
● Leaderboard with high scores
● Users can register an account to save progress
Species Guessing game
● The user is given a blurry image of an animal, sections of the image are unblurred over the course
of 1 minute
● Score is calculated based on how fast the user can guess before full image is revealed
● Time trials mode
Random Zoo Cam
● Users can cycle through random live wildlife/trail cam feeds from around the world
